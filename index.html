Contenidos del curso

Conocimientos básicos de Python, numpy y Pandas Vamos a armar un ambiente de desarrollo y ayudarte a dar los primeros pasos en la programación con Python para análisis de datos.

Recolección de datos Todo proyecto tiene un inicio y cuando de datos se trata es siempre uno muy desordenado. Vamos a aprender sobre como recolectar datos de distintas fuentes y formas, limpiarlos y transformarlos en estructuras para trabajar.

Análisis, visualización y presentación de datos Vamos a darte las herramientas para entender los datos y las mejores prácticas para presentarlos a tu audiencia.
